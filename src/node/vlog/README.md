# Vlog Backend

## Description

Uses [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation
1. Install nvm
1. Open terminal to this directory and run `nvm use`
1. Run the install: `yarn install`
1. Optionally, install VSCode Rest Client Plugin

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
