### Copying
Go inside this directory and run `rm -rf node_modules/ && tar cvf - . | (cd /dest/dir; tar xvf -)`

### Setup
1. Install hnvm
2. Run yarn add

### Start development
1. Run yarn start

### Build production
1. Run yarn build

### Setup react router
- Install dependencies: `yarn add react-router-dom @types/react-router-dom`
- [tutorial](https://www.robinwieruch.de/react-router/)
